# Dot Link Puzzle

자바스크립트 DOM Grid 기반의 간단한 닷 링크 퍼즐 MVP입니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다.

## 현재 구현

- 5x5 DOM Grid
- 같은 색 점끼리 드래그 연결
- 상하좌우 인접 칸 이동만 허용
- 다른 색 점/선과 충돌 방지
- 자기 경로 되돌아가기/잘라내기
- 전체 색 연결 완료 시 클리어 메시지
- 마우스/터치 공통 Pointer Events 사용

## 다음 개선 후보

- 모든 칸 채우기 조건 추가
- 레벨 자동 생성기
- SVG/Canvas 기반 선 렌더링
- 교육용 변형: 식-답, 단어-뜻, 그림-낱말 연결


## 업데이트

- 1단계부터 5단계까지 난이도를 다시 구성했습니다.
- Flow Free 방식처럼 모든 색을 연결하는 것뿐 아니라 모든 칸을 채워야 클리어됩니다.
- 1~3단계는 5x5, 4~5단계는 6x6입니다.


## 수정 메모

- 4~5단계처럼 `size: 6`인 레벨에서는 보드가 자동으로 6x6으로 확장됩니다.
- 보드 크기는 `currentLevel.size`를 기준으로 `width`, `height`, `--grid-size`를 함께 갱신합니다.
