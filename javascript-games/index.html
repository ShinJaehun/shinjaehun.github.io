4학년 수학 게임

평면도형의 이동

4학년 수학에서 배우는 평면도형의 이동을 게임으로 연습합니다.

점 이동하기

닷링크

같은 색 점을 연결하며 점이 어떻게 이동하는지 살펴보는 퍼즐입니다. 선을 겹치지 않게 그리고 모든 칸을 채우며 이동 경로를 생각합니다.

도형 밀기

러시아워

자동차 모양 도형을 위, 아래, 왼쪽, 오른쪽으로 밀어 봅니다. 다른 도형을 피해 빨간 자동차를 출구까지 이동시키는 게임입니다.

도형 돌리기 · 뒤집기

테트리스

떨어지는 블록을 움직이고 돌리며 빈 공간에 맞춥니다. 도형을 회전하거나 방향을 바꾸었을 때의 모양을 생각해 볼 수 있습니다.

평면도형 이동 응용하기

파이프드림

파이프 조각을 놓아 물이 지나갈 길을 만듭니다. 도형의 방향과 연결 관계를 생각하며 이동 경로를 완성합니다.

평면도형 이동 응용하기

파이프드림 2

제한 시간 안에 파이프를 배치하고, 시간이 끝나면 물이 흐릅니다. 기존 파이프를 바꾸며 더 나은 이동 경로를 만들어 봅니다.
