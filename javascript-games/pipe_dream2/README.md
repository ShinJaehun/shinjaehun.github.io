# 파이프 드림 2

업로드한 Pipe Dream 게임의 흐름을 바탕으로 새로 정리한 Canvas JavaScript 버전입니다.

## 실행

`index.html`을 브라우저에서 열면 됩니다.

## 게임 규칙

- 제한 시간 동안 파이프를 놓습니다.
- 시간이 끝나면 물이 자동으로 흐릅니다.
- 물이 정해진 시간 동안 끊기지 않고 계속 흐르면 성공입니다.
- 빈 칸을 누르면 현재 표시된 다음 파이프가 놓입니다.
- 이미 파이프가 있는 칸을 다시 누르면 기존 파이프가 깨지고 새 파이프로 바뀝니다.
- 물이 흐르기 시작하면 더 이상 파이프를 바꿀 수 없습니다.

## 파일

- `index.html`: 화면 구조
- `style.css`: UI 스타일
- `game.js`: 게임 로직
